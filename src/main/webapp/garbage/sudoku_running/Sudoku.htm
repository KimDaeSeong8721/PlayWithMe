Instructions

  • 지우기 버튼을 누르고 수동으로 스도쿠 '선물'을 입력하세요. 셀을 클릭하거나 키보드 화살표 키를 사용하여 '1'에서 '9' 사이의 숫자를 입력하면 됩니다. Delete 키 또는 '0' 키를 누르면 셀이 지워집니다. 숫자를 입력하거나 보드 위의 숫자 행을 클릭하여 지울 수도 있습니다.
  • 완료되면 로드 버튼을 누르십시오. 이제 주어진 것이 파란색으로 표시됩니다.
  • 재설정 버튼을 누르면 이제 모든 변경 사항이 취소되고 원래 상태로 돌아갑니다.
  • 허용되는 숫자는 각 셀의 배경에 작은 밝은 회색 숫자로 표시됩니다. '허용된 표시' 옵션을 선택 해제하여 이 기능을 끌 수 있습니다.
  • 허용된 숫자가 가능한 유일한 숫자인 경우(단일이라고 함) 빨간색으로 강조 표시됩니다. '싱글 보기' 옵션을 선택 해제하여 이 기능을 끌 수 있습니다.
  • 수락 버튼을 누르면 싱글이 실제 입력된 숫자로 변환됩니다.
  • 힌트 버튼은 현재 선택된 셀을 올바른 숫자로 채울 것입니다.
  • 숫자가 입력된 셀을 클릭하면 동일한 숫자가 빨간색으로 강조 표시됩니다.
  • 물론 해결은 전체 퍼즐을 풀 것입니다.
  • 실행 취소는 숫자를 입력하거나 수락 버튼을 누르는 것과 같은 마지막 사용자 작업을 실행 취소합니다.
